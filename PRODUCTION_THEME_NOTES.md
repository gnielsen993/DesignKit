# Production Theme Notes (2026-02-27)

Updated DesignKit palette from warm/upscale styling to a cleaner production baseline:

- Neutral, high-contrast surfaces (light + dark)
- More standard semantic colors for success/warning/danger
- Modern accent mapping per preset (teal/blue/rose/amber/slate)
- Better readability for day-to-day product use

## Intent
Preserve preset variety while making default UI feel app-store production friendly rather than boutique/editorial.
